xialeistudio.github.io
======================

xialei's blog

基于Angularjs+Bootstrap开发的博客，所有文章全部来自本人博客主站

##技术实现##
1.wordpress进行文章发布/编辑等操作
2.通过本人基于ThinkPHP 3.2.2 开发的一套接口系统（暂时只能读取文章和评论）给前端提供Ajax接口
3.angularjs读取接口数据

##特点##
+ 完全响应式，多终端兼容
+ 整站数据全部Ajax化
+ 采用CORS代替JSONP实现跨域（JSONP不支持POST）
